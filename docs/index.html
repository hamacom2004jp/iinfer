  • 物体検知
    • １．物体検知モデル(mmdetection)の環境をインストールする（通常Ubuntu上で実行）
    • ２．物体検知モデルをデプロイする（通常Windowsから実行）
    • ３．物体検知を実行する（通常Windowsから実行）
    • ４．検知したbboxをフィルターする（通常Windowsから実行）
    • ５．検知したbboxの内容から画像判定する（通常Windowsから実行）
    • ６．画像判定をCSV形式で出力する（通常Windowsから実行）
  • 領域検知
    • １．領域検知モデル(segmentation)の環境をインストールする（通常Ubuntu上で実行）
    • ２．領域検知モデルをデプロイする（通常Windowsから実行）
    • ３．領域検知を実行する（通常Windowsから実行）
    • ４．検知した領域をフィルターする（通常Windowsから実行）
    • ５．検知した領域の内容から画像判定する（通常Windowsから実行）
    • ６．画像判定をCSV形式で出力する（通常Windowsから実行）
  • 顔認識
    • １．顔認識モデルの環境をインストールする（通常Ubuntu上で実行）
    • ２．顔認識モデルをデプロイする（通常Windows上で実行）
    • ３．顔認識ストアを編集及びデプロイする（通常Windows上で実行）
    • ４．顔認識を実行する（通常Windows上で実行）
